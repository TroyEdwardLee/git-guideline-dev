# ESlint

`npm install -g eslint-cli`

## [Command Line Interface](https://eslint.org/docs/user-guide/command-line-interface)

`eslint [options] [file|dir|glob]*`

- `eslint "webapp/**/*.js"`
- `eslint --fix "webapp/**/*.js"`
